## 주요 기능
### HOME
- 웹페이지 메인 화면

### LOGIN 로그인 
- 로그인 시 세션에 저장

### JOIN 회원가입
- 회원 정보 저장

### BOARD 게시판
- 게시판 기능
- 게시글 전체 조회 기능
- 카테고리별 조회 기능

### BOARD -> Write 글 작성
- 카테고리, 제목, 내용을 저장하는 기능

### BOARD -> 상세글 조회
- 게시물 번호, 게시물 제목 클릭시 상세글 페이지로 이동
